# Googleスプレッドシート連携 ＆ 画像差し込みガイド

本ドキュメントでは、問い合わせフォームの送信データをGoogleスプレッドシートに自動蓄積するための設定手順と、各セクションの画像差し込み用プロンプト案について解説します。

---

## 1. Googleスプレッドシート連携手順

この設定を行うことで、LPの問い合わせフォームから送信されたデータがリアルタイムでGoogleスプレッドシートに自動追記されるようになります。

### ステップ 1: スプレッドシートの準備
1. [Googleスプレッドシート](https://sheets.google.com)を新規に作成します。
2. 1行目（A1セル〜G1セル）に、以下の見出しを入力します。
   - **A列**: 送信日時
   - **B列**: お名前
   - **C列**: フリガナ
   - **D列**: メールアドレス
   - **E列**: 電話番号
   - **F列**: 希望プラン
   - **G列**: お問い合わせ内容

### ステップ 2: Google Apps Script (GAS) の記述
1. スプレッドシートの上部メニューの **「拡張機能」** ＞ **「Apps Script」** をクリックします。
2. 開いたエディタ内に最初からあるコードをすべて消去し、以下のコードをコピー＆ペーストします。

```javascript
function doPost(e) {
  try {
    // データを保存するスプレッドシートを開く
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    
    // 受信したデータを解析
    var rawData = e.postData.contents;
    var data = JSON.parse(rawData);
    
    // 現在の日時を取得
    var timestamp = new Date();
    var formattedDate = Utilities.formatDate(timestamp, "Asia/Tokyo", "yyyy-MM-dd HH:mm:ss");
    
    // スプレッドシートに追加する行の配列を作成
    var row = [
      formattedDate,
      data.お名前 || "",
      data.フリガナ || "",
      data.メールアドレス || "",
      data.電話番号 || "",
      data.希望プラン || "",
      data.お問合せ内容 || ""
    ];
    
    // スプレッドシートの最終行にデータを追加
    sheet.appendRow(row);
    
    // CORS対応の返却値を作成して完了を通知
    return ContentService.createTextOutput(JSON.stringify({ status: "success" }))
      .setMimeType(ContentService.MimeType.JSON);
      
  } catch (error) {
    // エラー発生時のログ出力と返却
    return ContentService.createTextOutput(JSON.stringify({ status: "error", message: error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
```

3. ツールバーの **「保存（フロッピーディスクのアイコン）」** をクリックします。

### ステップ 3: Webアプリケーションとしてデプロイ
1. 画面右上の **「デプロイ」** ボタンをクリックし、**「新しいデプロイ」** を選択します。
2. 左側の「種類の選択」で **「ウェブアプリ」** を選択します（表示されていない場合は、歯車アイコンからウェブアプリを選びます）。
3. 各設定を以下のように指定します。
   - **説明**: 任意（例：問い合わせデータ受信用）
   - **次のユーザーとして実行**: 「自分（Your account）」
   - **アクセスできるユーザー**: **「全員（Anyone）」** （※「全員」にしないと、外部のLPからデータを送信できません）
4. **「デプロイ」** をクリックします。
5. 初回はGoogleから「アクセスの承認」を求められますので、アカウントを選択し、警告が出た場合は「詳細（Advanced）」＞「安全ではないページに移動（Go to ... (unsafely)）」を選択して、アクセスを承認（Allow）します。
6. デプロイ完了後、表示された **「ウェブアプリのURL」** をコピーします。

### ステップ 4: LPへのURL設定
1. 作成したプロジェクトの `スクリプト/メイン.js` を開きます。
2. 10行目付近にある `const スプレッドシート送信先URL = 'YOUR_GAS_WEBAPP_URL_HERE';` の部分を、コピーしたウェブアプリのURLに変更します。
   ```javascript
   // 例
   const スプレッドシート送信先URL = 'https://script.google.com/macros/s/XXXXX/exec';
   ```
3. ファイルを保存します。これで設定は完了です！

## 2. 差し込み画像＆プロンプトガイド

本LPでは画像ファイルを差し込まなくてもデザインが維持されるよう、CSSで上品なベージュ系グラデーションがプレースホルダーとして敷かれています。
画像をご用意のうえ、以下のファイル名（形式はPNG）で `画像/` フォルダへ配置していただくことで自動的に反映されます。

### ① メインビジュアル画像
- **ファイル名**: `メインビジュアル.png`
- **表示場所**: LPのファーストビュー背景
- **画像生成用プロンプト案**:
  > A premium and elegant main visual for a luxury Japanese beauty clinic. Clear skin, soft pink beige and warm gold bronze tone, minimalist background, natural lighting, high-end aesthetic. A beautiful close-up of a Japanese woman's radiant and glowing skin, smiling gently, expressing wellness, confidence, and self-care. Luxurious spa or modern clinic atmosphere. No text, high quality.

### ② クリニック内観・雰囲気画像

- **表示場所**: コンセプトセクション

Minimalist design, warm beige and bronze colors, indirect glowing warm lighting, elegant curved reception desk, marble floors, comfortable chic armchairs, soft shadows, serene and clean premium spa atmosphere, high-end design, photo.
